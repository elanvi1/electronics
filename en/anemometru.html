Anemometer

Factory calibrated digital anemometer. It locally stores data on a microSD card, has GSM / GPRS connectivity via a SIM card. Periodically uploads data on a FTP server. Can be configured by USB and remotely by SMS. Allows for alarms to be configured at wind overspeed. The dedicated PC software allows data visualization and interpretation. It is ideal for determining the fesability of installing wind turbines. Can be powered from grid or from small solar panel.
This solution can be adapted for any type of sensor: weather station, agriculture sensors (for soil, water), etc.
